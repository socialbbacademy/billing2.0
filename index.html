Google Sheets Sync

Connect Your Google Sheet

The app can send all entries directly to Google Sheets. After you paste the Apps Script web app URL, each add, delete, and clear action will sync the latest data to your sheet.

Use a Google Apps Script web app attached to your spreadsheet. The page sends JSON with all current entries, then the script rewrites the sheet from that latest data.

Firebase Sync

Sync Existing Data uploads the current browser data to Firebase in one snapshot. It includes work entries, salary records, expense records, and saved company names.

This sync uses the Firebase Realtime Database REST API and writes a full JSON backup into the `billingData` path.

No Firebase database URL saved yet.
Info
Sync sends these columns: ID, Date, Company, Work Description, Working Hour/Qty, Rate, Amount, Notes, File Name, Timestamp.
Apps Script Expected Behavior
Deploy a Google Apps Script web app that accepts a POST body with `action` and `entries`, clears the target sheet, writes the headers, then writes all rows again.
